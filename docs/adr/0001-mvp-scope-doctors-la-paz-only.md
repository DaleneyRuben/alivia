# MVP scope: doctors only, La Paz only

The founder's original idea spanned many verticals (doctors, hospitals, psychologists, barbers) and was open on geography. We're narrowing the MVP to solo/small-practice doctors (any specialty, not hospitals) in La Paz only. Hospitals require multi-department/multi-resource scheduling and an enterprise sales motion (procurement, IT gatekeepers) that would slow down validation; spreading across many cities or many unrelated verticals would leave every one of them too thin to build trust with patients. Doctors are chosen specifically because the founder has personal contacts there to bootstrap the first cohort, and specialty-filtered search (e.g. "cardiólogo") still feels useful to a patient even with a small number of doctors listed, unlike a generic multi-vertical directory.

Other verticals (barbers, psychologists as a distinct category, hospitals) and other cities remain candidates for expansion once the core loop is proven in La Paz.
